📋 SMTP 설정 가이드

Supabase Dashboard에서 SMTP 설정 확인

  1. Supabase Dashboard 접속
  2. Settings → Authentication 클릭
  3. SMTP Settings 섹션 확인
  4. SMTP 서버 정보가 올바르게 설정되어 있는지 확인
  5. Test SMTP 버튼으로 설정 테스트
⚠️ 주의사항

SMTP 설정이 완료되지 않으면 이메일이 발송되지 않습니다.
Gmail, Outlook 등의 SMTP 서버를 사용하거나 SendGrid, Mailgun 등의 서비스를 연동해야 합니다.
